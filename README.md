# ESP32 Knight

## Technology Stack

- Hardware: Nodemcu-32S
- VSCode
- PlatformIO

> [!TIP]
>
> Nodemcu-32S Not Support breakpoint at debug mode.

## esp32-knight-mcurl

Use UART to execute CURL commands, such as downloading data from HTTPS.

## esp32-knight-serial

Use Macro to package Serial messages.
